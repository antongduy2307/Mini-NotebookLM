# Examples

These files are synthetic and safe for demos.

Recommended demo:

1. Start the app with `uv run app`.
2. Create a workspace.
3. Upload `sample_notes.md`.
4. Build the workspace index.
5. Import `sample_eval_cases.json`.
6. Edit the imported cases and select the uploaded sample document.
7. Run retrieval evaluation.

The examples contain no API keys, private data, or copyrighted PDFs.
